# ATOS-jenkins-coderepo
Example Code Repo for Atos delivery

Launch instance
- Compute Engine
  - set name to: yourinitials-jenkins
  - Set Region
  - type: e2-small
  - Network tags: jenkins
  - boot disk: Ubunti 20.04 LTS

Open SSH in Browser

```bash
nano installjenkins
chmod +x installjenkins
./installjenkins
```
